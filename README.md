Calculator

A simple Python calculator that handles basic arithmetic without crashing on invalid input.

Usage

Run the script:

```bash
python calculator.py
```
Type operations like 4 + 3, -2.5 * 3, or 5 ^ 2.
Type quit to exit.

Supported operators
+, -, *, /, ^ (exponent).

Error handling

Rejects malformed input (shows an example of correct format).

Prevents division by zero.

Catches unknown operators and unexpected errors.

The program can handle all types of errors and It will not crash.
